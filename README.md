# Sharanya K R Portfolio

A personal portfolio website built using React and Vite to showcase projects, skills, certifications, and achievements.

## Features

- Responsive portfolio design
- Hero section with introduction and profile photo
- Resume download option
- Projects with live demo and GitHub links
- Certifications and NCC achievements
- Skills and education details
- Contact section with email and phone

## Setup

1. Open a terminal in your project folder `c:\PORTFOLIO`
2. Install dependencies:

```bash
npm install
```

3. Start the development server:

```bash
npm run dev
```

4. Open the local URL shown in the terminal.

## Customize Content

- Update `src/App.jsx` with your projects, certifications, and achievements
- Replace `public/resumekr.pdf` with your latest resume
- Replace `public/photo.jpeg` with your profile photo
